Q1. What is APT in Linux?  
A: A package management system for Debian-based distributions.

Q2. What does apt-get update do?  
A: It refreshes the local package index.

Q3. Why should update be run before installing packages?  
A: To ensure the latest versions are installed.

Q4. How do you search for packages using APT?  
A: Using apt-cache search <keyword>.

Q5. How do you install a package?  
A: sudo apt-get install <package>.

Q6. How does APT handle dependencies?  
A: It resolves and installs them automatically.

Q7. How did you verify Vim installation?  
A: Using vim --version.

Q8. What does apt-get remove do?  
A: Removes a package but keeps configuration files.

Q9. What is apt-get autoremove used for?  
A: Removing unused dependencies.

Q10. Why is package management important for security?  
A: It keeps systems updated and reduces vulnerabilities.
