Q1. What is awk used for in Linux?
A: awk is used for text processing, data extraction, and pattern scanning.

Q2. Why is awk suitable for structured data?
A: Because it automatically splits lines into fields.

Q3. How does awk reference columns?
A: Using $1, $2, $3, etc.

Q4. What does $0 represent in awk?
A: The entire line.

Q5. What does awk '{print $1, $3}' do?
A: Prints the first and third columns.

Q6. How do you filter rows using awk?
A: By applying conditions inside the awk expression.

Q7. What does awk '$2 > 25 {print $0}' do?
A: Prints rows where the second column is greater than 25.

Q8. Does awk work with space-separated files by default?
A: Yes.

Q9. Why is awk useful in automation scripts?
A: It processes data quickly without manual parsing.

Q10. Where is awk commonly used in real systems?
A: Log analysis, reporting, monitoring, and data processing tasks.
