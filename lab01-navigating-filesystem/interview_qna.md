**Q1. What does the `pwd` command do?**  
A: It displays the absolute path of the current working directory.

**Q2. Why is `pwd` important in security labs?**  
A: It prevents running commands in the wrong directory, which can cause security issues.

**Q3. What does the `ls` command show?**  
A: It lists files and directories in the current directory.

**Q4. What is the difference between `ls` and `ls -la`?**  
A: `ls -la` shows hidden files along with permissions and ownership.

**Q5. What does `cd ..` do?**  
A: It moves one level up to the parent directory.

**Q6. Why is directory navigation critical for system administrators?**  
A: Logs, configs, and scripts depend on correct filesystem paths.

**Q7. What are hidden files in Linux?**  
A: Files starting with a dot (.) used mainly for configuration.

**Q8. What does `/home/toor` represent?**  
A: The home directory of the user `toor`.

**Q9. How do file permissions help in security?**  
A: They define who can read, write, or execute files.

**Q10. Where are these commands used in real-world jobs?**  
A: In SOC analysis, cloud servers, Linux administration, and automation.
