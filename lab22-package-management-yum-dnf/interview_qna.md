Q1. What is YUM in Linux?  
A: A package manager for RPM-based systems like CentOS and RHEL.

Q2. What does yum update do?  
A: Refreshes metadata and applies available updates.

Q3. What is the difference between yum update and yum upgrade?  
A: Upgrade may remove obsolete packages.

Q4. How do you search for a package using YUM?  
A: yum search <keyword>.

Q5. How did you install Vim in this lab?  
A: Using sudo yum install vim-enhanced.

Q6. How does YUM handle dependencies?  
A: Automatically resolves and installs them.

Q7. How did you verify Vim installation?  
A: By running vim --version.

Q8. How do you remove an installed package?  
A: sudo yum remove <package>.

Q9. What is DNF?  
A: The modern replacement for YUM on newer RPM-based systems.

Q10. Why is package management critical in enterprises?  
A: It ensures consistency, security, and system stability.
