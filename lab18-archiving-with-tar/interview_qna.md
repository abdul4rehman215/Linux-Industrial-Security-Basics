Q1. What is tar used for in Linux?  
A: It bundles files and directories into a single archive.

Q2. What does tar -czf do?  
A: Creates a compressed tar archive using gzip.

Q3. What does the -c option mean?  
A: Create a new archive.

Q4. Why is -z used with tar?  
A: To compress the archive using gzip.

Q5. What does -f specify?  
A: The archive file name.

Q6. How do you list archive contents without extracting?  
A: Using tar -tzf archive.tar.gz.

Q7. What does -t do in tar?  
A: Lists files inside the archive.

Q8. How do you extract a tar.gz archive?  
A: Using tar -xzf archive.tar.gz.

Q9. Why is tar important for backups?  
A: It preserves directory structure and metadata.

Q10. Where is tar commonly used?  
A: Backups, log archiving, and software distribution.
