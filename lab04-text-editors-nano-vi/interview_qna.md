Q1. What is the purpose of text editors like nano and vi in Linux?  
A: They are used to create and edit configuration files, scripts, and documentation.

Q2. What happens when you open a non-existing file using nano?  
A: Nano creates the file automatically and opens it for editing.

Q3. Why is nano considered beginner-friendly?  
A: It is non-modal and displays shortcut keys on the screen.

Q4. How do you save and exit a file in nano?  
A: Press Ctrl + O to save, then Ctrl + X to exit.

Q5. In which mode does vi open a file by default?  
A: Vi opens files in Normal Mode.

Q6. How do you enter Insert Mode in vi?  
A: By pressing the i key.

Q7. Why is understanding vi modes important?  
A: Editing is only possible in Insert Mode, and commands differ by mode.

Q8. How do you save changes and exit vi?  
A: Press Esc, type :wq, and press Enter.

Q9. What is a key difference between nano and vi?  
A: Nano is simple and mode-less, while vi is modal and more powerful.

Q10. Why are text editors essential in system administration and security roles?  
A: They are required to edit system configs, scripts, logs, and service files.
