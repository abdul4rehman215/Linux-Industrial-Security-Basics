Q1. What are environment variables in Linux?  
A: Key-value pairs used to store configuration and runtime information.

Q2. How can you view all environment variables?  
A: Using the env command.

Q3. What format do environment variables follow?  
A: KEY=VALUE.

Q4. How do you create an environment variable?  
A: Using export VAR=value.

Q5. How do you access an environment variable?  
A: Using echo $VAR.

Q6. Why is export important?  
A: It makes variables available to child shells.

Q7. Why was MYNAME available in a new bash shell?  
A: Because it was exported.

Q8. How do you remove an environment variable?  
A: Using unset VAR.

Q9. What does blank output from echo $VAR indicate?  
A: The variable is unset or does not exist.

Q10. Where are environment variables commonly used?  
A: Shell scripting, automation, and application configuration.
