Q1. What does ifconfig display?  
A: Network interface configuration such as IP and MAC addresses.

Q2. Why is ip addr preferred over ifconfig?  
A: It is modern, actively maintained, and more detailed.

Q3. What interface was used in this lab?  
A: ens5.

Q4. What does ping test?  
A: Network connectivity using ICMP.

Q5. What does 0% packet loss indicate?  
A: Successful connectivity.

Q6. What protocol does ping use?  
A: ICMP.

Q7. What is nslookup used for?  
A: Resolving domain names to IP addresses.

Q8. How is dig different from nslookup?  
A: dig provides more detailed DNS output.

Q9. Why are DNS tools important?  
A: They help troubleshoot name resolution issues.

Q10. Where are these tools commonly used?  
A: Cloud environments, servers, and security investigations.
