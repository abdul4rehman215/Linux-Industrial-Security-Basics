Q1. What is an alias in Linux?  
A: A shortcut that maps a short command to a longer command.

Q2. Why are aliases useful?  
A: They reduce repetitive typing and improve efficiency.

Q3. How do you create an alias?  
A: Using the alias command.

Q4. What did the alias ll do in this lab?  
A: It executed ls -l.

Q5. Are aliases permanent by default?  
A: No, they are temporary for the current shell session.

Q6. How can you test if an alias works?  
A: By running the alias name and checking the output.

Q7. How do you remove an alias?  
A: Using the unalias command.
