Q1. What does the grep command do?
A: It searches for text patterns inside files or directories.

Q2. What does grep stand for?
A: Global Regular Expression Print.

Q3. How do you search for a word inside a file?
A: Using grep "word" filename.

Q4. What output does grep show by default?
A: Only the lines containing the matching pattern.

Q5. How do you search recursively using grep?
A: By using the -r option.

Q6. Why is recursive grep useful?
A: It allows searching across multiple files and directories at once.

Q7. What does grep -r "password" logs/ do?
A: Searches all files under logs for the word "password".

Q8. What does the -A option do in grep?
A: Displays lines after the matching line.

Q9. What does the -B option do in grep?
A: Displays lines before the matching line.

Q10. Where is grep commonly used in real-world jobs?
A: Log analysis, troubleshooting, security investigations, and monitoring.
