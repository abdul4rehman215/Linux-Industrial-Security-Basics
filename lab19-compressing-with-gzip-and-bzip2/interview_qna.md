Q1. What is file compression?  
A: It reduces file size to save storage and improve transfer efficiency.

Q2. What does gzip do?  
A: Compresses a file and replaces it with a .gz version.

Q3. How do you decompress a .gz file?  
A: Using the gunzip command.

Q4. What happens to the original file after gzip compression?  
A: It is replaced by the compressed file.

Q5. What does bzip2 do?  
A: Compresses files using higher compression ratios than gzip.

Q6. How do you decompress a .bz2 file?  
A: Using the bunzip2 command.

Q7. Which is faster: gzip or bzip2?  
A: gzip is faster.

Q8. Which provides better compression: gzip or bzip2?  
A: bzip2 provides better compression.

Q9. Why is compression important in backups?  
A: It saves storage space and reduces transfer time.

Q10. Where are gzip and bzip2 commonly used?  
A: Log compression, backups, and archive management.
