Q1. What is the purpose of the mkdir command in Linux?  
A: It is used to create new directories.

Q2. How do you verify that a directory was created successfully?  
A: By using the ls command to confirm it appears in the output.

Q3. How do you create a subdirectory inside an existing directory?  
A: By specifying the path, for example: mkdir Documents/sub_directory.

Q4. What does the rmdir command do?  
A: It removes empty directories only.

Q5. Why does rmdir fail on non-empty directories?  
A: For safety reasons, it does not delete directories containing files.

Q6. Which command removes directories with files inside them?  
A: rm -r, used carefully.

Q7. How can you rename a directory in Linux?  
A: By using mv old_name new_name.

Q8. How do you move a directory into another directory?  
A: By using mv directory_name target_directory/.

Q9. What is the difference between renaming and moving a directory?  
A: Renaming changes the name, moving changes the location.

Q10. Why are directory management commands important in security roles?  
A: They help organize data, manage access, and avoid accidental misplacement of files.
