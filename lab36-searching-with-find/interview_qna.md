Q1. What is the find command used for?
A: It searches files and directories recursively based on conditions.

Q2. Is find recursive by default?
A: Yes, it searches through all subdirectories automatically.

Q3. How do you find all .txt files?
A: Using find -name "*.txt".

Q4. What does the -size option do?
A: Filters files based on size.

Q5. What does +1M mean in find?
A: Files larger than 1 megabyte.

Q6. How do you execute commands on found files?
A: Using the -exec option.

Q7. What does {} represent in -exec?
A: The current file being processed.

Q8. Why is find important for system administrators?
A: It helps audit, clean, and manage files efficiently.

Q9. Can find be used in automation scripts?
A: Yes, it is commonly used in scripts.

Q10. Where is find used in cybersecurity?
A: File discovery, sensitive file audits, and incident response.
