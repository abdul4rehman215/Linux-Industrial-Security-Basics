Q1. What is the purpose of top?
A: To display real-time system and process information.

Q2. Is top installed by default?
A: Yes, on most Linux systems.

Q3. What is htop?
A: An enhanced, interactive system monitoring tool.

Q4. How do you install htop on Ubuntu?
A: sudo apt install htop.

Q5. How do you exit top or htop?
A: Press q.

Q6. How do you sort by CPU usage in top?
A: Press P.

Q7. How do you sort by memory usage in top?
A: Press M.

Q8. How do you sort processes in htop?
A: Press F6 and select the column.

Q9. Which tool supports mouse interaction?
A: htop.

Q10. Why is monitoring important in production systems?
A: To detect performance issues, failures, and security incidents early.
