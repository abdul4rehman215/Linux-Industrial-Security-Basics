Q1. What is the purpose of the head command in Linux?  
A: It displays the beginning lines of a file.

Q2. How do you view a specific number of lines using head?  
A: By using the -n option, for example head -n 5 filename.

Q3. What does the tail command do?  
A: It displays the last lines of a file.

Q4. Where is tail commonly used in real environments?  
A: For monitoring logs and recent system activity.

Q5. When should you use more instead of cat?  
A: When viewing large files that require paginated output.

Q6. What keys are used to navigate in more?  
A: Space for next page, Enter for next line, q to quit.

Q7. What is the role of the grep command?  
A: It searches for text patterns inside files.

Q8. What does grep -i do?  
A: Performs a case-insensitive search.

Q9. What does grep -r do?  
A: Searches recursively through directories.

Q10. Why are file-viewing commands important in security roles?  
A: They are essential for analyzing logs and configuration files.
